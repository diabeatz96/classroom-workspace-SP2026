# Practice Problems - Topic 02: Variables and Input

---

## Problem 1: Temperature Display (Basic)

Write a program that declares and displays three temperature values:
- A cold temperature (-10)
- Room temperature (72)
- A hot temperature (98.6)

Use appropriate data types for each.

---

## Problem 2: Simple Calculator Input (Basic)

Write a program that:
1. Asks the user to enter two numbers
2. Displays both numbers back to them

**Example:**
```
Enter first number: 5
Enter second number: 10
You entered 5 and 10
```

---

## Problem 3: Student Profile (Intermediate)

Write a program that collects and displays student information:
- First name (single word)
- Age
- GPA
- Major (full name, may contain spaces)

Remember: Use the correct approach for reading strings with spaces.

---

## Problem 4: Unit Conversion Setup (Intermediate)

Write a program that:
1. Asks the user for their height in inches (can be decimal, like 67.5)
2. Asks for their weight in pounds
3. Displays the values with labels

This sets up for arithmetic in the next topic.

---

## Problem 5: Mad Libs (Challenging)

Create a simple Mad Libs game that asks for:
- A name
- An adjective
- A noun
- A verb (past tense)
- A place

Then displays a silly sentence using all the words.

Example output:
```
Enter a name: Bob
Enter an adjective: fuzzy
Enter a noun: banana
Enter a verb (past tense): jumped
Enter a place: the moon

Bob was feeling fuzzy when a banana jumped all the way to the moon!
```
