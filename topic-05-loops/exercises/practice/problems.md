# Practice Problems - Topic 05: Loops

---

## Problem 1: Print Multiples (Basic)

Write a program that prints the first 10 multiples of a number entered by the user.

**Example:**
```
Enter a number: 7
7 14 21 28 35 42 49 56 63 70
```

---

## Problem 2: Factorial (Basic)

Write a program that calculates the factorial of a number.
(n! = n * (n-1) * (n-2) * ... * 1)

**Example:**
```
Enter a number: 5
5! = 120
```

---

## Problem 3: Input Validation (Intermediate)

Write a program that keeps asking for a number until the user enters a number between 1 and 100 (inclusive).

**Example:**
```
Enter a number (1-100): 150
Invalid. Try again.
Enter a number (1-100): -5
Invalid. Try again.
Enter a number (1-100): 42
You entered: 42
```

---

## Problem 4: Average Calculator (Intermediate)

Write a program that reads numbers until the user enters -1, then displays the average of all numbers entered (not including -1).

**Example:**
```
Enter numbers (-1 to stop): 10
Enter numbers (-1 to stop): 20
Enter numbers (-1 to stop): 30
Enter numbers (-1 to stop): -1
Average: 20
```

---

## Problem 5: Number Pyramid (Challenging)

Write a program that prints a number pyramid of height n.

**Example (n=4):**
```
   1
  121
 12321
1234321
```

---

## Problem 6: Prime Number Check (Challenging)

Write a program that determines if a number is prime. A prime number is only divisible by 1 and itself.

**Example:**
```
Enter a number: 17
17 is prime.

Enter a number: 15
15 is not prime.
```
