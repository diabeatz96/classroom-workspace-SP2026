# Practice Problems - Topic 03: Arithmetic and Expressions

---

## Problem 1: Circle Calculations (Basic)

Write a program that:
1. Asks the user for the radius of a circle
2. Calculates and displays the area and circumference

Use 3.14159 for pi.
- Area = pi * r * r
- Circumference = 2 * pi * r

---

## Problem 2: Temperature Converter (Basic)

Write a program that converts Fahrenheit to Celsius:
- Formula: C = (F - 32) * 5 / 9

Make sure you handle the division correctly!

---

## Problem 3: Time Breakdown (Intermediate)

Write a program that:
1. Asks the user for a number of seconds
2. Converts it to hours, minutes, and remaining seconds

Example: 3725 seconds = 1 hour, 2 minutes, 5 seconds

Hint: Use integer division and modulo.

---

## Problem 4: Grade Average (Intermediate)

Write a program that:
1. Asks for 5 test scores
2. Calculates the average (as a decimal)
3. Displays the average with 2 decimal places

Remember: Watch out for integer division!

---

## Problem 5: Money Breakdown (Challenging)

Write a program that:
1. Asks for a total amount in cents
2. Calculates the minimum number of quarters, dimes, nickels, and pennies needed

Example: 87 cents = 3 quarters, 1 dime, 0 nickels, 2 pennies
