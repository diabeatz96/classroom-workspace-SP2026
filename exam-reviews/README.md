## Study Tips

1. **Review worked examples** from each topic
2. **Practice coding by hand** - exams may require writing code on paper
3. **Trace through code** - practice following program execution step by step
4. **Understand concepts** - do not just memorize syntax
5. **Do practice problems** - especially ones you found challenging
6. **Review your mini-projects** - make sure you understand all the code you wrote
