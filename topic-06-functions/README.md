# Topic 06: Functions

## Overview

In this topic, you will learn how to organize code into reusable, modular pieces called functions. Functions make programs easier to write, test, debug, and maintain.

## What You Will Learn

- Function definition and calling
- Parameters and arguments
- Return values
- Void functions
- Function prototypes
- Variable scope (local vs global)
- Pass by value vs pass by reference

## Key Concepts at a Glance

| Concept | Example |
|---------|---------|
| Function with return | `int add(int a, int b) { return a + b; }` |
| Void function | `void printHello() { cout << "Hello"; }` |
| Function call | `int result = add(5, 3);` |
| Pass by reference | `void swap(int &a, int &b)` |
| Prototype | `int add(int a, int b);` |
