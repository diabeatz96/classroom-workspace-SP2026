# Topic 01: Output and Program Structure

## Overview

In this topic, you will learn how to display text and create your first complete C++ programs. This is the foundation for everything else you will learn in this course.

## What You Will Learn

- How to display text using `cout`
- Escape sequences for special characters
- Comments to document your code
- The structure of a C++ program
- How to compile and run programs

|---------|---------|
