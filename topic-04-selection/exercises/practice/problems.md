# Practice Problems - Topic 04: Selection Structures

---

## Problem 1: Even or Odd (Basic)

Write a program that asks the user for a number and displays whether it is even or odd.

**Example:**
```
Enter a number: 7
7 is odd.
```

---

## Problem 2: Voting Eligibility (Basic)

Write a program that asks for the user's age and determines if they are old enough to vote (18 or older).

**Example:**
```
Enter your age: 17
You are not old enough to vote.
```

---

## Problem 3: Largest of Three (Intermediate)

Write a program that asks for three numbers and displays the largest one.

**Example:**
```
Enter three numbers: 5 12 8
The largest is 12.
```

---

## Problem 4: Leap Year (Intermediate)

Write a program that determines if a year is a leap year.
- A year is a leap year if divisible by 4
- EXCEPT years divisible by 100 are NOT leap years
- EXCEPT years divisible by 400 ARE leap years

**Examples:**
```
Enter a year: 2024
2024 is a leap year.

Enter a year: 1900
1900 is not a leap year.

Enter a year: 2000
2000 is a leap year.
```

## Problem 5: Simple Calculator (Intermediate)

Write a program that asks for two numbers and an operator (+, -, *, /), then displays the result. Use a switch statement.

**Example:**
```
Enter first number: 10
Enter operator (+, -, *, /): *
Enter second number: 5
10 * 5 = 50
```

## Problem 6: Shipping Cost (Challenging)

Calculate shipping cost based on weight and destination:
- Local (L): $2.00 base + $0.50 per pound
- Regional (R): $5.00 base + $0.75 per pound
- National (N): $10.00 base + $1.00 per pound

**Example:**
```
Enter package weight (lbs): 3
Enter destination (L/R/N): R
Shipping cost: $7.25
```

## Problem 7: BMI Calculator (Challenging)

Calculate BMI and display the category:
- BMI = weight (lbs) * 703 / height (inches) squared
- Underweight: < 18.5
- Normal: 18.5 - 24.9
- Overweight: 25 - 29.9
- Obese: 30+

**Example:**
```
Enter weight (lbs): 150
Enter height (inches): 67
BMI: 23.5
Category: Normal
```
